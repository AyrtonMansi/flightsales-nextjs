# FlightSales.au

Australia's premier aircraft marketplace. Built with Next.js 14.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

Push to GitHub and connect to Vercel, or:

```bash
npx vercel
```

## Tech Stack

- Next.js 14
- React 18
- Plus Jakarta Sans typography
- Uber-clean black/white design system
